# Tic Tac Toe Game 

This is a simple Tic Tac Toe game built using HTML, CSS and JavaScript.

## Features
- Two player mode
- CPU mode (easy)
- Reset game option
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap

## How to Run
1. Download or clone the repository
2. Open index.html in your browser
